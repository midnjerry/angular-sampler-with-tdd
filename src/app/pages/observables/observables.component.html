Observables

Observables can be considered streams of data. They only stream data if there is an Observer. Once an observer is assigned aka subscribed, then the Observable starts pushing data by calling Observer.next(data);

To bind an Observer with an Observable you call the Observable's subscribe(...) method and pass in the observer as a parameter. When the Observable gets new data, it will push that value to the Observer by calling it's .next(data) method. One aspect of Observables that can be considered confusing is this .next method.
