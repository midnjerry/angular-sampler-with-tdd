Map

Map is a new data structure introduced in ES6. It allows you store key-value pairs.

let map = new Map();

Add / Get / Delete Entries

  • map.set(key, value)
  • map.get(key) - returns value at given key or null
  • map.has(key) - returns true if map contains key
  • map.delete(key) - deletes key,value pair from map
  • map.size - returns number of key/value pairs in map
  • map.clear() - clears all entries
