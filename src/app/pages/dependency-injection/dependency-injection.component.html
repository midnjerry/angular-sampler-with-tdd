Dependency Injection

Dependency Injection is the technique in which an object receives other objects that it depends on.

Dependency Injection is an extremely important topic. You will see it being discussed in relation to frameworks and testing. I highly recommend you thoroughly understand dependency injection before trying to learn Spring Boot or Angular. This is what you gain by understanding dependency injection:

  • It's easier to design classes for testing.
  • It's easier to learn Spring Boot / Angular.
  • It's easier to debug Spring Boot / Angular issues.

We are going to learn dependency injection by studying each keyword found within its definition.

  • Object
  • Dependency
  • Injection

By the end of this lesson, you should be able to understand what dependency injection is from a code standpoint and, you should be able to identify the different techniques used to inject a dependency. Let's get started!

What is an object?

In computer science, an object can be a variable, a data structure, a function, or a method, and as such, is a value in memory referenced by an identifier. In Java and Javascript, an object refers to a particular instance of a class, where the object can be a combination of variables, functions, and data structures.
